# 🚀 Junechecktt

> Modern TikTok profile lookup interface with premium glassmorphism design

---

## ✨ Overview

**Junechecktt** là nền tảng tra cứu thông tin tài khoản TikTok với giao diện hiện đại, lấy cảm hứng từ iOS và thiết kế glassmorphism cao cấp.

Hỗ trợ nhập nhanh:
- `@username`
- Link TikTok
- `user_id`

---

## ⚡ Features

- 🔍 Tra cứu hồ sơ TikTok nhanh chóng  
- 🎯 Hỗ trợ nhiều định dạng input  
- 💎 UI glassmorphism cao cấp  
- 📱 Responsive hoàn hảo (mobile-first)  
- ⚙️ Tối ưu trải nghiệm đọc thông tin
